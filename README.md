# 🏛️ RoboExhibit

RoboExhibit is an AI-powered interactive museum assistant that combines:

- natural language interaction
- ontology-based knowledge retrieval
- adaptive visitor support in virtual environments

## 👥 Team

- Antonio Spedito
- Alessandro Picone
- Lucrezia Mosca

## ✨ Key Components

- **Knowledge Layer:** RDF ontology (`Ontologia.rdf`)
- **Reasoning / Querying:** SPARQL-based information retrieval
- **Language Layer:** LLM-driven response generation
- **Backend API:** `app.py` for request handling and pipeline orchestration
- **Unity Integration:** immersive museum side (shared separately)

## 🚀 Running the backend

This repo includes the backend entrypoint in `app.py`.  
Install project dependencies in your Python environment and run:

```bash
python app.py
```
